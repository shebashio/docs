## Overview

k9s is a terminal-based Kubernetes management tool that provides a rich interactive UI for navigating clusters, viewing resources, and performing common operations. It is pre-installed in DEDZED Kasm workspaces and is the recommended way to interact with your ephemeral clusters.

---

## Launching k9s

```bash
# Launch k9s with the current context
k9s

# Launch k9s with a specific context
k9s --context <context-name>

# Launch k9s targeting a specific namespace
k9s -n bigbang
```

<Tip>
After [connecting to your cluster](/kasm-workspaces/connect-cluster), run `k9s` in your terminal to start exploring resources immediately.
</Tip>

## Basic navigation

| Action | Key |
|--------|-----|
| Move down / up | `j` / `k` or arrow keys |
| Select resource | `Enter` |
| Go back | `Esc` |
| Quit | `q` or `Ctrl+C` |

---

## Resource views

Switch resource view by pressing `:` and typing the resource abbreviation or name.

### Common resource shortcuts

| Command    | Resource           |
| ---------- | ------------------ |
| `:po`      | Pods               |
| `:dp`      | Deployments        |
| `:svc`     | Services           |
| `:cm`      | ConfigMaps         |
| `:ing`     | Ingresses          |
| `:ns`      | Namespaces         |
| `:pv`      | PersistentVolumes  |
| `:secret`  | Secrets            |
| `:no`      | Nodes              |
| `:ds`      | DaemonSets         |
| `:sts`     | StatefulSets       |
| `:cj`      | CronJobs           |
| `:pvc`     | PersistentVolumeClaims |

Press `Tab` after `:` to see all available resource types.

## Filtering and searching

| Action | Key |
|--------|-----|
| Filter resources | `/` then type query (supports regex) |
| Clear filter | `Esc` or `Ctrl+u` |
| Search across resources | `Ctrl+f` |

## Viewing resource details

| Action | Key |
|--------|-----|
| Describe resource | `d` |
| View logs | `l` (Pods only) |
| Follow logs | `Shift+f` (in logs view) |
| View YAML manifest | `y` |
| View metrics | `0` (in Pods view) |

---

## Resource actions

| Action | Key |
|--------|-----|
| Edit resource | `e` (opens in default editor) |
| Delete resource | `Ctrl+d` |
| Scale deployment up / down | `+` / `-` |
| Execute command in Pod | `x` |
| Port forwarding | `Shift+f` |
| Save YAML to file | `s` |

## Namespace and context management

| Action | Key |
|--------|-----|
| Switch namespace | `n` |
| View all namespaces | `Shift+a` |
| Switch contexts | `:ctx` |

## Complete key bindings

### Navigation

| Key          | Action                           |
| ------------ | -------------------------------- |
| `j` / `k`    | Move down / up                   |
| `h` / `l`    | Move left / right or collapse / expand |
| `g`          | Go to top                        |
| `G`          | Go to bottom                     |
| `Ctrl+f` / `Ctrl+b` | Page down / up           |

### Resource interaction

| Key          | Action                     |
| ------------ | -------------------------- |
| `/`          | Filter resources           |
| `d`          | Describe selected resource |
| `l`          | View logs                  |
| `e`          | Edit resource              |
| `y`          | View YAML                  |
| `s`          | Save YAML to a file        |
| `Ctrl+d`     | Delete resource            |
| `x`          | Execute command in Pod     |
| `Shift+f`    | Port forwarding            |
| `+` / `-`    | Scale deployment up / down |

### General

| Key          | Action                          |
| ------------ | ------------------------------- |
| `:`          | Command mode (enter resource shortcuts) |
| `Esc`        | Back or clear filter            |
| `Ctrl+r`     | Refresh view                    |
| `?`          | Help menu                       |
| `Ctrl+a`     | Select all                      |
| `q` / `Ctrl+C` | Quit k9s                     |

## Tips and tricks

- **Customize k9s**: Edit `~/.k9s/config.yml` to customize themes and settings.
- **Copy to clipboard**: Press `c` in supported views.
- **Toggle wide view**: Press `w` to toggle wide columns for more detail.
- **View previous commands**: Press `Ctrl+p` to see command history.
- **Bookmark resources**: Press `Shift+b` to bookmark, and `b` to view bookmarks.
- **Sort columns**: Press `Shift+<number>` to sort by column.
