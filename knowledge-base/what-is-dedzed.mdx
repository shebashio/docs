## Digital Environment and Development for Zero-trust Enablement and Data

### What is DEDZED?

DEDZED stands for **Digital Environment and Development for Zero-trust Enablement and Data**. It is an IL5 (Impact Level 5) environment designed for rapid, secure technology assessment and software development. Key features and purposes of DEDZED include:

- Providing a secure IL5 environment for testing complex web-based software builds
- Facilitating collaborative software development with zero-trust access
- Offering comprehensive AWS and Azure Government environments with GitOps integration for seamless application deployment and management
- Integrating advanced scanning solutions and software supply-chain tools to ensure code security and compliance
- Serving as a precursor to production software deployment, assisting developers in producing fast test results
- Providing a secure, browser-based Virtual Desktop Infrastructure (VDI) solution with zero-trust capabilities

DEDZED aims to accelerate development velocity, enhance security, and provide a cost-effective experience for software development in high-security environments.
